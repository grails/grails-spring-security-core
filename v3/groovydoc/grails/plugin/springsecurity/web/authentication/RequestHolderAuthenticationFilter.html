[Groovy] Class RequestHolderAuthenticationFilter

          • grails.plugin.springsecurity.web.authentication.RequestHolderAuthenticationFilter

          • Extends the default org.springframework.security.web.authentication.UsernamePasswordAuthenticationFilter to store the request and response in the SecurityRequestHolder.

            deprecated:
            will be removed and replaced with grails.plugin.springsecurity.web.SecurityRequestHolderFilter at the beginning of the filter chain
